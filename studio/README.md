# Sanity Studio for MovingWalls Website

This is the Sanity Studio for managing content on the MovingWalls website.

## Setup

1. Install dependencies:
```bash
cd studio
npm install
```

2. Run the studio locally:
```bash
npm run dev
```

3. Deploy to Sanity.io:
```bash
npm run deploy
```

The studio will be available at: https://movingwalls-website.sanity.studio/

## Content Types

- **Blog Posts** - Company blog articles
- **Case Studies** - Client success stories
- **Press Releases** - News and press coverage
- **Events** - Conferences, webinars, and meetups
- **Landing Pages** - Custom marketing pages (page builder)
- **Authors** - Content creators
- **Categories** - Blog/content categories
